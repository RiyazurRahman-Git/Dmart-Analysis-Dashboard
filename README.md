# 📊 Dmart Retail Performance Analysis 🚀

This repository contains a **Power BI dashboard** analyzing Dmart’s retail performance, payments 💳, categories 🛒, seasonal trends 📆, and customer insights 👥.  
It includes interactive visuals, datasets, reports, and presentation slides 🎨 for storytelling.

---

## 📂 Project Contents
- 🖥️ `Dmart Analysis.pbix` → Interactive Power BI dashboard  
- 📊 `Dmart Sales Report.xlsx` → Source dataset for analysis  
- 📑 `Dmart Analysis.pdf` → Exported PDF version of the dashboard  
- 🖼️ `Dmart Analysis DOC/` → Presentation images

- Power BI ⚡
